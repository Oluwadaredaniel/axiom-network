## Issues

If you encounter a bug or have a feature request:

1. Search [existing issues](https://github.com/Emerald-dev0/axiom-network/issues) to avoid duplicates
2. Use the appropriate issue template (bug report or feature request)
3. Provide as much context as possible

## Community

- **Discussions:** Use GitHub Discussions for questions and ideas
- **Security:** See [SECURITY.md](SECURITY.md) for reporting vulnerabilities

## Before Asking

1. Are you on the latest version? Try `git pull`
2. Have you checked the `.env.example` for missing configuration?
3. Have you run `npm run setup` to initialize the database?
4. Check the FAQ in the [README](README.md)
