# Investor Review: Axiom

## 1. Market Opportunity
**The "Machine Customer" Economy.** Gartner predicts that by 2028, machines will represent trillions in digital commerce. Axiom is positioned to be the primary gateway for these transactions.

## 2. Business Model
- **Transaction Fees:** 2-5% on every AXC settlement.
- **SaaS for Enterprise:** Private registries for secure internal agent economies.
- **Provider Commissions:** Lead generation fees for top-tier capability providers.

## 3. Competition
- **Traditional APIs:** Too much friction (KYC, static keys, subscriptions).
- **Stripe/PayPal:** Designed for humans; high fees for micro-transactions.
- **OpenAI/Anthropic:** Walled gardens. Axiom is an **Open Protocol** for all providers.

## 4. Defensibility (The Moat)
- **Network Effects:** As more providers join, the registry becomes more valuable to orchestrators.
- **Trust Moat:** Our **Reputation Data** is proprietary and hard to replicate. An agent with a 100/100 Axiom score has a portable "credit score" for the AI world.
- **Protocol Lock-in:** Integration into the Conductor/SDK level makes switching costs high.

## 5. Weaknesses & Risks
- **LLM Reliability:** We rely on underlying model providers for planning.
- **Regulation:** Potential future "KYC for Agents" requirements.
- **Adoption:** Requires a shift in how developers build agents (from static to dynamic).

## 6. Long-term Potential
Axiom could become the **Standard Ledger** for all machine-to-machine interactions, including GPU sharing, data training, and specialized logic.
